Installing Python

The latest stable release is Python 3.13.1 (December 3, 2024). To install on Windows 10:

  1. Open a terminal: Press Win + R, type cmd, and press Enter.
  2. Check if Python is installed: Type python3 --version and press Enter.
  3. If not installed, type python3 to open the web installer and click "Get."
  4. Verify installation by running python3 --version again.
